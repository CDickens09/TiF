FAQ Page

Here you can find instructions on how to use the BlackBoard question pool maker as well as some general information about the tool.

What is this tool for?

This tool is designed to be used to aid in creating unique question sets for classes on Blackboard. That is to say a series of questions that only differ in a numerical sense.

Do the inputs require coding?

No inputs require specific coding knowledge and will work with just typed text. However, to get the most out of the tool several languages have been built in to aid in laying out questions. This includes a built in Markdown conversion as well as the ability to type in raw HTML code and MathJax commands.

Can I use my own variables?

Yes, the system has been designed to accept specific values in the form of a CSV file. This must be laid out in a specific way to ensure the values you want end up in the desired question. Please see the Example page for more information on this.

Can I use images in questions?

Yes, you can use any image that can be retrieved from the interent. These can also be viewed in the preview to show how the image will look next to any text. To include images you will need to use either HTML or Markdown as shown on the Example page.
