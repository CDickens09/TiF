Blackboard Upload:

Pool Title:

To choose your pool title (Which is useful for identifying your pool in Blackboard once you have several) simply type in the pool title box whatever you want the title to be.

Setting-up Variables:

Random Variables:

To setup random variables click on the "Random Variables" tab on the top bar, you will be presented with the below screen:
Once on this screen you can select the number of variables you would like to use with either the arrows or by typing in the desired number. When you have selected a number betwen one and ten the options for each variable will be displayed.
Each variable will be presented to you with four options; "Variable Placeholder", (The name of your variable which will be needed between {{}} to use them) "Min", (The minimum value for the variable) "Max", (The maximum value for the variable) "Decimals". (The number of decimals for the variable)
The bottom box allows you to choose the number of questions you wish to generate with random variables. (This is the number of questions that will be outputted in the final pool if the variable selector is set to random)

Random Variables:

To use a random variable in the question or feedback boxes place the variable placholder from the option box on the random variable tab inside {{Placeholder}}. For example in the image below the variable "a" is used by typing {{a}}.
Similarly variables can be used in the numerical answer and error boxes, but in these boxes mathematical operators can also be used such as * or +.
