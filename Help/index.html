Here you can find instructions on how to use the BlackBoard question pool maker.

What is this tool for?

This tool is designed to be used to aid in creating unique question sets for classes on Blackboard. That is to say a series of questions that only differ in a numerical sense. Options are changeable so that you can give unique questions to each student while ensuring the answer and question are generated in tandom to avoid any calculation error.
