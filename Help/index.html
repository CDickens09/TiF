Help Page

Here you can find instructions on how to use the BlackBoard question pool maker.

What is this tool for?

This tool is designed to be used to aid in creating unique question sets for classes on Blackboard. That is to say a series of questions that only differ in a numerical sense.

Do the inputs require coding?

No inputs require specific coding knowledge and will work with just typed text. However, to get the most out of the tool several languages have been built in to aid in laying out questions. This includes a built in markdown conversion as well as the ability to type in raw html code (Allows for images to be put in with tags).
